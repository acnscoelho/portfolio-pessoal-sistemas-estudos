Backup e Restauração

📤 Fazer Backup

Faça o download de um arquivo com todos os seus dados para ter uma cópia de segurança.

📥 Restaurar Backup

Carregue um arquivo de backup para restaurar seus dados. Atenção: Isso substituirá todos os dados atuais!

🗑️ Limpar Dados

Remove todos os dados salvos da aplicação. Use com cuidado!

ℹ️ Informações sobre Backup

  • Formato: Arquivo JSON com extensão .json
  • Conteúdo: Todas as matérias, tópicos, configurações e progresso
  • Compatibilidade: Funciona entre diferentes navegadores e computadores
  • Segurança: Dados ficam apenas no seu computador
  • Recomendação: Faça backup regularmente, especialmente antes de limpar o navegador
